# 📊 Student Performance Analysis (Excel Project)

## 📌 Overview
This project analyses student performance using Microsoft Excel. The objective is to identify patterns between study hours and exam scores and provide data-driven insights.

## 📊 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts (Bar, Pie, Scatter)
- Statistical Functions (AVERAGE, CORREL, COUNTIF)

## 🔍 Key Analysis
- Descriptive statistics (mean, min, max)
- Performance classification using IF function
- Correlation analysis between study hours and exam scores
- Data visualisation using charts
- Pivot table analysis

## 📈 Key Insights
- Students who study more tend to achieve higher scores
- A significant proportion of students are high performers
- Positive relationship between study hours and exam performance

- Data cleaning and preparation
- Data analysis using Excel
- Data visualization
- Analytical thinking
